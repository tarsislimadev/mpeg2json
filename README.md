# mpeg2json

Speech to text, from MPEG to JSON files

## how to install

Install using [Node.js Package Manager]()

```bash
npm i mpeg2json -g
```

## how to run

Export your `GEMINI API KEY`

```bash
export GEMINI_API_KEY=
```

Run `mpeg2json` 

```bash
mpeg2json ./video.mp4
```

It'll create a `video.json` file
